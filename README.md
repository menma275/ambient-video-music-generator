# Ambient Video Music Generator

（アンビエント・ビデオ・ミュージック・ジェネレーター）

Webカメラの映像からリアルタイムに色彩情報を抽出し、アンビエント（環境音楽）を生成するWebアプリケーションです。

## 仕組み

本アプリケーションは、映像の「色」を音楽の「構成要素」に変換することで、その場の空気感を音に変換します。

### 1. 映像解析 (Vision)

- Webカメラの映像から5つのサンプリングポイントを監視します。
- 各ポイントのピクセルデータ（RGB）を取得し、**HSV（色相・彩度・明度）**に変換します。
- これにより、色の「種類」「鮮やかさ」「明るさ」を音楽のパラメータとして利用可能にします。

### 2. 音声生成 (Audio)

Web Audio APIを使用して、以下の処理を行っています。

- **不協和音の排除**: 抽出された色相（Hue）は、Cメジャー・ペンタトニックスケール（ド・レ・ミ・ソ・ラ）のいずれかの音にマッピングされます。これにより、どのような映像であっても調和の取れた音楽になります。
- **音色の質感**: 柔らかい三角形波（Triangle wave）をベースに、ADSRエンベロープで長いアタックとリリースを設定し、音が静かに現れて消えていく質感を演出しています。
- **空間エフェクト**:
  - **Reverb**: ホワイトノイズから生成したインパルス応答を用いたコンボルバー。深い残響を生み出します。
  - **Delay**: フィードバックループを持つディレイ。音を重層的に積み上げます。
  - **Filter**: 彩度（Saturation）に応じてカットオフ周波数を変化させ、音の「こもり具合」や「明るさ」を制御します。

### 3. マッピングルール

映像の各要素は以下のように音に変換されます：

- **色相 (Hue)**: 音程（スケール内のどの音を鳴らすか）
- **彩度 (Saturation)**: 音の明るさ（フィルターの開き具合）および音量
- **明度 (Value)**: オクターブ（高い音か低い音か）

### 4. アルゴリズム

5つのサンプリングポイントは、それぞれ独立したランダムな周期（3〜8秒程度）で発音します。このポリリズム（多重リズム）的なアプローチにより、音楽が単調な繰り返しにならず、常に変化し続ける有機的なサウンドスケープが形成されます。

## セットアップと実行

1. `ambient/index.html` をブラウザ（Chrome, Edge, Safari推奨）で開きます。
2. カメラのアクセス許可を求められたら「許可」を選択してください。
3. 「Start Music」ボタンをクリックすると生成が開始されます。

## 技術スタック

- HTML5 Canvas API
- Web MediaDevices API (Webcam)
- Web Audio API (Oscillator, Gain, Convolver, Delay, BiquadFilter)
- Vanilla JavaScript
